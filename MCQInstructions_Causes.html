Detailed Instructions

In each example, you are given a prompt, such as:

Selling a new book causes name exposure. When is this possible?

or

Selling new book causes name exposure. When is this impossible?

you should complete the prompt with the first reasonable thing that comes to your mind. For example:

Selling new book causes name exposure. When is this possible?
readers like your book

Selling new book causes name exposure. When is this impossible?
the author is anonymized

In case the prompt sentence does not make sense to you, or you cannot think of any good responses, click on the checkbox with label "It does not make sense".


Our goal is to capture the most common perceptions that come to your mind and is specific to each prompt. Please make sure your answers are specific, concise and usual. A simple way to write the answers is to focus on the mental image you have after reading the prompt, and then describe what makes the task possible or impossible to occur. Below, we have compiled a short list of common bad responses and how to fix them:

  1. Avoid Pronouns when referring to objects or people. In the examples below, for the "book", "author", and "seller" use their name explicitly when referring to them.
    Selling new book causes name exposure. When is this possible?
    • It is a bestseller.
    • the book is a bestseller.
    Selling new book causes name exposure. When is this impossible?
    • The author is using a pen name.
    This rule also applies when referring to the person(s)performing the task,
    Getting drunk typically causes slurred speech. When is this impossible?
    • You pass out.
    • The person passes out.
  2. Do Not Overthink: Avoid using unusual responses. We are looking for things that are obvious to you but not mentioned in the prompt.
      Selling new book causes name exposure. When is this impossible?
    • All the people are dead.
    • No one can read.
  3. Be Specific: Avoid using general responses that are not specific to the prompt. Additionally, please do not use responses that can be used for other prompts no matter the situation.
      Selling new book causes name exposure. When is this impossible?
    • The book is bad.
  4. Avoid Negatives: Avoid using negatives (e.g. not) in your responses.
      Selling new book causes name exposure. When is this possible?
    • The author is not using a pen name.
  5. Write actual sentences: Avoid using ungrammatical, or broken english.
      Selling new book causes name exposure. When is this impossible?
    • Bad book
